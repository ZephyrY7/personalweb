Noise Filter Application

This is an assignment being done for Digital Signal Processing module.

This project focuses on developing a Speech Enhancement System using MATLAB to improve speech signal quality by reducing noise interference. It addresses challenges posed by both external and internal noises that often degrade communication quality.

The primary outcome is the significant enhancement of speech quality, with reduced noise interference.

Features

  • • Noise classification input
  • • Different filter types (Butterworth, Chebyshev, Elliptic, and Kaiser Window)
  • • Graphical output representation

This Speech Enhancement System showcases the effectiveness of digital signal processing in improving communication clarity, with potential applications in audio technology advancements.

The project repo can be found on noise filter.
