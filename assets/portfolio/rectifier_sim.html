Rectifier Simulation Application

This is an assignment being done for Engineering Software & Application module.

This project focuses on the development of an educational Power Converter Simulator application using MATLAB App Designer. The App Designer simplifies app development with its interactive UI elements, streamlining the creation process and enhancing accessibility for users.

The simulator elucidates the functionalities of basic power converter circuits, specifically half-wave and full-wave rectifiers. It demonstrates the effects of rectification with and without a capacitor filter, offering a hands-on learning experience.

Features

  • • Adjustable input
  • • Graphical comparison
  • • Ability to save simulated results

The Power Converter Simulator serves as a practical educational tool, merging theoretical concepts with interactive simulations. It provides a comprehensive understanding of rectifier operations and the impact of capacitor filters on signal waveforms.
