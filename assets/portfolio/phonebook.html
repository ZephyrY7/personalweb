Phonebook

This is an assignment being done for C Programming module.

This project is on developing a Telephone Directory system using C programming, tailored for a university setting. The system aims to provide a functional tool for managing employee information within the university.

Features

  • • Data management (addition, storage, & retrieval)
  • • Search Functionality
  • • Update and Delete

The system streamlines the process of managing employee contact details, making it more efficient. This Telephone Directory system serves as a practical demonstration of the application of C programming in creating a useful tool for organizational data management, specifically tailored for university environments.

The project repo can be found on phonebook.
