UniHero - Concept Design

This is a project being done for iTank USM 2023.

UniHero is an application designed to promote recycling and sustainable practices within university campuses.

UniHero features a bonus point system for recycling, community engagement with society activities, and a platform for student projects to request recyclable materials. It also includes a scanner to identify recyclable items, promoting sustainable practices.

UniHero is set to promote recycling efforts on university campuses, making sustainability accessible and rewarding for all.
