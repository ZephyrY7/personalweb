Manufacturing Process Simulation Application

This is an assignment being done for Engineering Software & Application module.

The project focuses on developing a simulation tool for a Lotion Manufacturing Industry using LabView. This tool emulates the operational flows of a factory, providing a virtual representation of the manufacturing process.

Features

  • • GUI with input selections on manufacturing
  • • Process simulations
  • • Event logging

The simulation replicates the lotion manufacturing process, providing insights into operational aspects. It serves as an educational tool for understanding chemical manufacturing and offers a platform for further development and customization.

The project repo can be found on manufacturing process simulation.
