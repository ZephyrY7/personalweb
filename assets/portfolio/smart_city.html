Smart City

This is an assignment being done for Microprocessor system & embedded software module.

The project described in this paper focuses on the development of a Smart City system, integrating advanced technology to enhance urban living. Key features include a Smart Traffic Light System, Autonomous Solar Tracking System, Smart Waste Management System, and Smart Street Lighting System. Implemented using an Arduino MCU and programmed in C, the project aims to create a more efficient, sustainable, and convenient urban environment, aligning with the broader goals of smart cities.

Each group member contributed to different components of the smart city design, which were later integrated into a integrated system.

The project repo can be found on arduino.
