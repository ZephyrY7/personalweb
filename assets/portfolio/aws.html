Website deployment - AWS

This is an assignment being done for Cloud infrastructure & services module.

A website deployed using Amazon Web Services (AWS), creating a secure, scalable, and efficient platform for managing global development statistics. Leveraging AWS services like EC2, S3, RDS, and auto-scaling, the website ensures high availability and performance. Collaboration tools like GitHub and Microsoft Teams were used for efficient development.

The project also involved setting up various AWS services, configuring security groups, and ensuring the website can handle traffic spikes. The outcome is a robust website that provides a seamless user experience and enhances data management and security.
