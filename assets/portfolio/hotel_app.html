Hotel Application - UI Design

This is an assignment being done for Human Computer Interaction module.

This project is on developing a mobile application for The RuMa Hotel and Residences to enhance the digital experience for guests, streamlining the hotel's services.

Features

  • • Room booking system
  • • Digital room key
  • • Cashless payment on parking

The project aims to create a user-friendly hotel management system that provides convenience, efficiency, and an environmentally friendly approach to hotel services. It aims to offer a superior guest experience by leveraging technology to meet modern-day hospitality needs.
